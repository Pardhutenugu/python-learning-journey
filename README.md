## Topics Covered

- Variables  
- Data Types: `int`, `float`, `string`, `boolean`  
- Input & Output: `print()` and `input()`

- Comparison Operators: `==`, `!=`, `<`, `>`, `<=`, `>=`  
- Logical Operators: `and`, `or`, `not`  
- Strings: concatenation, repetition, indexing, slicing, methods  
- Conditional Statements: `if / else`  
- Boolean Logic: `True / False`  
- Practice Exercises  

## Notes

- Variables store data in memory.  
- Python is dynamically typed; no need to declare variable types.  
- Strings are immutable.  
- Python indices start at 0.  
- User input is always a string; convert to numbers with `int()` or `float()` if needed.  
- Conditional statements allow decision-making.  
- Boolean values are `True` or `False`.  
- Logical operators help combine conditions.  
- Practicing examples strengthens understanding.
